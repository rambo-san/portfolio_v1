 # Interactive Developer Portfolio & Arcade

A high-performance developer portfolio built with Next.js, showcasing systems engineering skills through interactive browser-based games.

## Tech Stack

- **Frontend**: Next.js 15 (App Router), TypeScript, Tailwind CSS
- **Animations**: Framer Motion
- **Gaming**: HTML5 Canvas (No heavy engines unless required)
- **Styling**: Custom Glassmorphism System

## Project Structure

- `src/components/layout`: Global navigation and footer.
- `src/components/sections`: Landing page sections (Hero, About, Projects).
- `src/components/ui`: Reusable UI tokens (Buttons, Cards).
- `src/app/arcade`: The gaming hub.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Design System

The project uses a dark, "cyber-premium" aesthetic with:
- Deep space backgrounds (`#030014`)
- Vibrant accents (Cyan, Violet, Pink)
- Glassmorphism for cards and overlays
- Smooth refined animations suited for high-end feel.
